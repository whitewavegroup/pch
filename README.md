# ProConnectHub / MiContacto (pch)

Bilingual directory MVP for entertainment & events (EN/ES), with premium UI,
sponsor banner, pricing (demo mode), directory filters, and refined provider page.

## Quick Start
```bash
npm install
npm run dev
# open http://localhost:3000
```

## Pages
- `/` – Home (Hero, Categories, Sponsor, Pricing, Testimonials)
- `/directory` – Filters by Category/Subcategory/Location with sample listings (PR, FL, Bogotá)
- `/provider/[id]` – Refined provider profile page

## Admin demo toggles (on Home page header > Admin)
- Lead Credits: OFF by default
- Demo Mode: ON by default (upgrades apply instantly)

## Tailwind
Configured via `tailwind.config.js` and `styles/globals.css`.
